Legend:

STO: Stores the onscreen number into STO_VAR.

STO_VAR: Inputs the stored number onto the screen.

DEL: Deletes the latest character on the screen.

CRL: Refreshes the calculator back to '0' ending all stacked operations.

√: Performs the square root operation.

Xy: Raises the current number to the power of an inputted number.

X2: Raises the current number to the power of 2.

"dot" : Inputs a decimal to the onscreen number.

+ {add}, - {subtract}, * {multiply}, / {divide}

Technical restrictions:

The onsrceen character limit is 7 characters.

An output of an answer over 7 characters will be returned in scientific notation.

NaN: The results of the numbers inputed into the calculator is undefined or not a number.

The calculator will enter an error stage if operations are inputted incorrectly.
